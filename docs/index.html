How to Make Changes

This dashboard is read-only. To add, edit, or delete teams, business groups, or APIs:

  1. Click the "Request Changes" button above
  2. Choose the appropriate template (Add Team, Add Business Group, etc.)
  3. Fill out the form and submit the issue
  4. Our workflow will automatically process your request and create a pull request

Note: All changes require approval through pull requests for security and compliance.

Dashboard is Read-Only

This dashboard displays the current state of your platform configuration. To make changes:

  • Add/Edit Teams: Use the "Add New Team" issue template
  • Add/Edit Business Groups: Use the "Add New Business Group" issue template
  • Add/Edit APIs: Use the "Add New API" or "Reassign API" issue templates
  • Delete Teams: Use the "Delete Team" issue template

All changes go through our automated workflow and require pull request approval.
